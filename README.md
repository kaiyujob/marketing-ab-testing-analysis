# Marketing A/B Testing and Conversion Analysis

This project analyzes the performance of marketing campaigns using A/B testing, statistical hypothesis testing, and funnel analysis.

The goal was to determine whether a test campaign improved user engagement and purchase outcomes compared with a control campaign.

## Tools and Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy

## Analysis Performed

- Data cleaning and preprocessing
- KPI analysis
- Funnel analysis
- Statistical hypothesis testing
- Confidence interval analysis
- Trend analysis
- Correlation heatmaps

## Key Findings

- The test campaign significantly improved CTR and user engagement.
- However, conversion efficiency declined.
- Purchase outcomes showed no statistically significant improvement.
- Higher engagement did not translate into stronger business value.
- Add-to-cart behavior was more predictive of purchases than clicks.
